# About this project

#### This is FTP server and client, client using PyQt GUI framework

## Dependdencies
* PyQt5.x
* pyftpdlib
* python3.x

## Usage
```bash
$ ./server.py
$ ./gui_client.py
```

>Note :
Current default port in server.py is 2121. If you use default FTP port 20 and 21, you may need to use sudo.

## Platform
Linux
